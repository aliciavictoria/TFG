\chapter{Repositorio del proyecto}
\label{anexo:repositorio}

El código desarrollado durante este trabajo de fin de grado se encuentra disponible en el siguiente repositorio: \url{https://github.com/aliciavictoria/LINGUINI-LLM-PROMPTING}

El repositorio incluye los scripts principales del sistema experimental, el código de ejecución de las estrategias, el módulo de extracción y recálculo de métricas, así como las instrucciones necesarias para descargar el benchmark Linguini, configurar las claves de API y reproducir los experimentos descritos en la memoria.
